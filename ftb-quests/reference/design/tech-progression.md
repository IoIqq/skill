# Tech Mod Progression Design Guide

## Common Task Flow Patterns

### Linear Resource Chain
Gather raw material → Smelt/process → Craft component → Build machine → Unlock next tier

Example (Create):
- Mine copper ore → Smelt copper ingot → Craft copper sheet → Build mechanical press

### Parallel Specialization
Two paths that converge:
- Path A: Mechanical power (windmill → shaft → cogwheel → gearbox)
- Path B: Fluid processing (pump → pipe → tank → mixer)
- Convergence: Automated assembly line needs both

### Escalating Complexity
Each tier adds one new concept:
1. Tier 1: Manual crafting (basic items)
2. Tier 2: Simple automation (deployers, mechanical crafters)
3. Tier 3: Multi-step processing (crushing, washing, smelting chain)
4. Tier 4: Full automation (logistics, monitoring, feedback loops)

## Mod-Specific Progression Tips

### Create
- Start with hand tools → windmill/waterwheel → mechanical processes
- Key bottleneck: Andesite Alloy (requires both mining and crafting)
- Late game: trains, contraptions, automated factories
- Common task types: item (crafting), observation (seeing a contraption work)

### Mekanism
- Start with basic machines → digital miner → fusion reactor
- Key bottleneck: Osmium (needed for most machines)
- Ore processing tiers: basic → advanced → elite → ultimate
- Common task types: item (ore processing), energy (power generation)

### Thermal Series
- Start with basic machines → dynamos → augment system
- Key bottleneck: Invar (requires iron + nickel)
- Focus on augment upgrades as milestone tasks
- Common task types: item (crafting augments), fluid (molten metals)

### EnderIO
- Start with basic capacitors → alloys → conduits
- Key bottleneck: Vibrant Alloy (requires ender pearls)
- Soul vials and spawners as advanced content
- Common task types: item (alloy crafting), kill (mob farm tasks)

### Applied Energistics 2
- Start with certus quartz → presses → processors → storage
- Key bottleneck: Meteorites (requires exploration)
- Progression: basic storage → autocrafting → complex patterns
- Common task types: item (processor crafting), observation (network visualization)

### Immersive Engineering
- Start with hemp → fiber → fabric → engineer's manual
- Key bottleneck: Steel (requires blast furnace, coke oven chain)
- Heavy machinery as visual milestones
- Common task types: item (multiblock components), location (building multiblocks)
