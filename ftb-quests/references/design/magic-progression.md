# Magic Mod Progression Design Guide

## Common Task Flow Patterns

### Discovery-Based
Explore → Find resource → Unlock recipe → Craft tool → Discover new mechanic

Example (Botania):
- Find mystical flower → Craft petals → Make lexicon → Build mana spreader

### Ritual Chain
Each ritual unlocks the next tier:
1. Simple ritual (basic resource)
2. Infusion ritual (enchanted items)
3. Complex ritual (boss summons or artifacts)

### Knowledge Gate
Tasks that require reading/understanding before proceeding:
- Read guidebook → Craft book stand → Unlock chapter → Advanced recipes

## Design Principles for Magic Quests
- Use observation tasks to encourage experimentation
- Reward exploration over grinding
- Hidden tasks work well as easter eggs
- Description text should hint at mechanics, not just list requirements
- Use §d (light purple) and §5 (dark purple) for magic-themed text formatting
